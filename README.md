# StudySync

StudySync is a cloud-based task management web application designed to help students organize courses, manage personal tasks, and collaborate through shared tasks in one platform.

## Live Demo

https://studysync-database.web.app

## Features

- User authentication
- Personal task management
- Shared tasks
- Course organization
- User profile page
- Real-time data synchronization
- Responsive dashboard interface

## Technologies Used

- React
- JavaScript
- CSS
- Firebase
- Firestore
- Vite
